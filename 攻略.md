# 🍄 Super Mario Bros. 完全攻略

---

## 🎮 操作说明

| 功能 | 按键 |
|------|------|
| 移动 | `←→` 方向键 / `A` `D` |
| 跳跃 | `Space` / `↑` / `W` |
| 蹲下 | `↓` / `S` |
| 发射火球 | `X` / `J` |
| 加速跑 | `Shift` |
| 暂停 | `P` / `Esc` |
| 重启关卡 | `R`（普通模式） |
| 选关 | 暂停时按 `1`-`5`（普通模式） |

> [!TIP]
> 火球为**自动追踪**，会自动瞄准最近敌人！范围250像素内全部精准命中。

---

## 🏠 游戏模式

### 普通模式 (Normal Mode)
- 每关2条命，999秒时间
- 过关后生命和时间**重置**
- 支持选关（暂停按1-5）和重新开始（R键）

### 极限模式 (Extreme Mode)
- 全程2条命，999秒共享
- **禁止选关、禁止重新开始**
- 真正的一命通关挑战

### World 1-1：草原 (Grassland)

**主题**：经典阳光草原 | **难度**：★☆☆☆☆

**关键道具**：
- 第16列问号砖：金币
- 第21列问号砖：**蘑菇/火焰花**（首个变身机会！）
- 第22列（第5行）高处问号砖：金币
- 第79列（第5行）问号砖：**蘑菇/火焰花**

**地形陷阱**：
- 第69-70列、第87-88列、第154列：断崖坑洞，注意不要掉落

**水管捷径**：
| 入口 | 出口 | 说明 |
|------|------|------|
| 第28列水管（↓蹲下进入） | 第163列出口 | 跳过大段关卡，直达后半部 |
| 第57列水管（↓蹲下进入） | 第179列出口 | 更大幅度的跳关 |

**地下密室**：水管进入后有大量金币砖块（第5行、第8行、第10行），满级刷金币的好去处！100金币 = 1UP。

**通关要点**：
1. 开场就拿第21列蘑菇，变大后可踩破砖块
2. 利用第1个水管（第28列）捷径可大幅缩短流程
3. 终点前有8层阶梯（第181-188列），从最高点跳到旗杆顶部得分最高
4. 旗杆位于第189列，跳得越高分数越高

| 平台跳跃 | 18-42 | 多层浮空平台，第39列有火焰花 |
| 垂直迷宫 | 45-58 | 纵向墙壁隔断，需要折返攀爬 |
| 死亡深渊 | 58-85 | 无底洞上方窄平台，钟乳石压迫 |
| 冲刺区 | 85-110 | 开阔地带但敌人密集 |
| 终极挑战 | 110-140 | 狭窄通道+阶梯通往出口 |

**关键道具**：
- 第39列（第6行）：**火焰花**
- 第51列（第6行）：**蘑菇/火焰花**
- 第94列（第7行）：**火焰花**（高处，需精确跳跃）

**水管捷径**：第42列水管 → 第128列出口（一口气跳过迷宫和深渊！）

**通关要点**：
1. 区域3迷宫：先向右跳到第49列平台，利用第9行中间平台登上第6行
2. 区域4死亡深渊：千万别贪，稳步跳跃不要冲太快
3. 水管捷径（第42列）极其推荐使用，可跳过最难的区域3-5

### World 1-3：空中竞技场 (Athletic)

**主题**：高空浮台 | **难度**：★★★★☆

**核心机制**：除起点和终点外**无地面**！全程在浮空平台间跳跃。

**平台路线**：
```
起点(地面0-15) → 浮台链 → 终点(地面140+)
每个平台宽2-6格，间距需精确跳跃
```

**关键道具**：
- 第14列（第7行）：**蘑菇/火焰花**
- 第44列（第6行）：**蘑菇/火焰花**
- 第82列（第5行）：**蘑菇/火焰花**
- 第115列（第3行）：**蘑菇/火焰花**（极高处，挑战性最大）

**踏石板**：地图中散布着单格石台（第6号方块），可作为紧急落脚点

**水管捷径**：第8列水管 → 第140列出口（直接跳到终点区域！）
- 地下密室有大量金币砖块（锯齿排列），非常划算

**通关要点**：
1. 开局从地面直接够到第8列水管可以一步到终点
2. 正常路线需按 Shift 加速才能跳过较远的平台间距
3. 中段平台越来越窄（最窄仅2格），脚滑的注意力要集中
4. 抓紧每个道具砖，变大后的跳跃距离更远

### World 1-4：城堡 (Castle)

**主题**：暗红色城堡 | **难度**：★★★★☆

**关卡特色**：岩浆深坑 + 钟乳石 + 狭窄通道

**岩浆坑分布**（掉下去即死）：
第21-22、41-42、56-57、66-67、79-80、93-94、106-107列

**关键道具**：
- 第11列（第9行）：**火焰花**（开局就能拿！）
- 第39列（第6行）：**蘑菇/火焰花**
- 第75列（第8行）：**蘑菇/火焰花**
- 第101列（第8行）：**蘑菇/火焰花**

**难点区域**：
1. **狭窄通道**（60-65列）：低天花板+墙壁隔断，走底部缝隙通过
2. **之字岩浆区**（81-92列）：需在平台间反复跳跃
3. **通道障壁**（112列）：有底部缺口，蹲下或小马力奥直接通过

**水管捷径**：第46列水管 → 第98列出口
- 城堡密室宝物丰厚：3排金币砖 + 2个火焰花方块！

**通关要点**：
1. 开场拿第11列火焰花，全程远程消灭敌人
2. 第46列水管密室是获取大量金币和火焰花的好机会
3. 注意岩浆坑间距，某些坑只有2格宽非常隐蔽
4. 终点前的8级阶梯（第126列起），从顶部跳到旗杆

### World 2-1：噩梦要塞 — Boss 战 (Bowser's Nightmare Fortress)

**主题**：终极Boss战 | **难度**：★★★★★

#### 路线总览

```
入口走廊(0-20) → 双层迷宫(21-70) → 地狱走廊(71-110) 
→ 深渊跳跃(111-140) → Boss战场(141-175) → 庆典大厅(176-220)
```

#### 区域详解

**区域1 — 入口走廊（0-20列）**
- 第6列：**火焰花**（必拿！Boss战生死攸关）
- 第10列、第14列：金币

**区域2 — 双层迷宫（21-70列）**
- **上层路线**（第7行空中浮桥）：通过左侧阶梯（22-27列）攀上
  - 第35列有金币，第50列有**火焰花**（上层路线的奖励）
  - 浮桥每7格有断裂，需跳跃
- **下层路线**：地面行走但有4个岩浆坑（29-30、39-40、49-50、59-60列）
- **推荐走上层**：敌人少、道具多、风险可控

**区域3 — 地狱走廊（71-110列）**
- 低天花板压迫走廊（75-82列、90-97列）
- 第84列、第101列有道具方块（金币和火焰花）
- 注意3格高石柱（第87列、第106列），需跳过

**区域4 — 深渊跳跃（111-140列）全段岩浆！**
- 6个浮空平台链，需精准跳跃
- **隐藏砖块**🔥：
  - 第115-116列（第12行）
  - 第119-120列（第11行）
  - 第127-128列（第12行）
  - 第132-133列（第10行）
  - 从下方头顶砖块会显现为固体踏石

> [!IMPORTANT]
> 掉入岩浆即死！隐藏砖块提供额外安全路径，但需要盲跳或提前知道位置。

**隐藏水管 — 星星密室**（第143列）：
- 在Boss区前方的地面蹲下（↓）进入
- 密室内有金币砖块 + **星星方块**（第5行第7列）
- ⚠️ **星星方块必须从上方踩踏才能获得星星，从下方顶会变成金币！**
- 还有火焰花（第5行第5列）

#### ⚔️ Boss 战 — 库巴 (Bowser)

**阶段1（桥上战斗）**：HP = 5
- 库巴在桥面（149-164列）巡逻
- 每1.8秒发射火球（朝马力奥方向）
- 随机跳跃

**攻击方式**：
| 方法 | 效果 |
|------|------|
| 火球命中 | HP -1 |
| 触碰斧头（166列） | 桥梁崩塌，2秒后进入阶段2 |
| HP降到0 | 直接进入阶段2（无需碰斧） |
| 星星爆炸 | HP -2 + 范围AOE |

> [!TIP]
> **最佳策略**：先用火球打到HP=0触发阶段2转换，或直接跑到166列碰斧头破桥。

**阶段2（狂暴模式）**：HP = 6
- 库巴全面复活，速度翻倍！
- **三连火球弹幕**：精确瞄准马力奥位置发射
- 追踪AI：每秒重新锁定马力奥方向
- 随机跳跃高度增大（4.0-6.5格）
- 岩浆不会杀死库巴（会浮在上面）

**阶段2攻略**：
1. 利用高台（第8行第152-153列、第6行第157-158列）躲避
2. 保持距离，用火球消耗HP
3. 注意三连弹幕时预判走位
4. 星星爆炸（8秒无敌结束时）可造成AOE 2点伤害

**击败后**：
- 第168列墙壁打开（门自动开启）
- 走到门前自动进入庆典大厅
- 领奖台 + 烟花 + 奖杯🏆
- 按 Enter 完成游戏通关！

## 🔑 隐藏要素

1. **每关水管密室**：所有关卡都有可进入的水管，内含大量金币和道具
2. **"王者荣耀"彩蛋**：1-2关卡第8列第2行问号砖
3. **隐藏砖块**（2-1关第4区域）：从下方头顶显现，提供岩浆上的安全路径
4. **星星密室**（2-1关第143列水管）：蹲下进入，获取星星无敌
5. **100金币1UP**：集满100枚任意金币获得额外生命
6. **顶砖杀敌**：大马力奥从下方顶砖块，可杀死站在砖块上方的敌人

## 🏆 高分技巧

1. **旗杆跳**：从最高点跳到旗杆，根据高度获得大量分数
2. **火球刷分**：火焰花的追踪火球远距离批量刷敌人
3. **龟壳连杀**：踢出龟壳消灭一排敌人，每个+100分
4. **密室刷币**：每个水管密室都有大量金币，有时间就进去
5. **Boss战高分**：击败库巴 = +10000分
6. **星星爆炸连杀**：无敌结束的范围爆炸可同时消灭多个敌人

## 🐛 彩蛋与秘密

- **观察者模式**（`B`键）：按住B键进入无碰撞飞行模式，自由浏览关卡地形
- **时间制度**：999秒实时倒计时，归零直接Game Over
- **马力奥受伤降级**：
  - 火焰状态 → 大马力奥（保留变大）
  - 大马力奥 → 小马力奥（附带0.5秒缩小动画）
  - 小马力奥 → 死亡
